
Instructions
============

To run you'll need the following modules: ```numpy, matplotlib, os, astropy, h5py``` as well as jupyter notebook (or lab).

Clone the repo:

```> git clone https://github.com/nealegibson/hr_desert_tutorial```

Then open up the tutorial notebook:

```> jupyter notebook simple_retrieval_tutorial.ipynb```

or 

```> jupyter lab simple_retrieval_tutorial.ipynb```
